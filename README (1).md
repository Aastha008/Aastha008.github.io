# Aastha Gupta — Portfolio

A 3D interactive personal portfolio site, built as a single static HTML file with a Three.js hero scene and CSS-driven hover interactions.

**Live site:** [aastha008.github.io](https://aastha008.github.io)

## About

I'm a 3rd-year B.Tech Computer Science & Engineering (IoT) student at VIT Vellore, with a BS in Data Science from IIT Madras. This site is where I showcase my work across IoT/embedded systems, machine learning, and full-stack web development.

## Features

- **3D animated hero** — a live Three.js IoT mesh network (orbiting sensor nodes connected to a central hub), with mouse parallax
- **3D tilt interactions** — project and contact cards respond to cursor position with a perspective tilt and glow effect
- **Sidebar navigation** — a slide-in menu with quick links and contact details
- Fully responsive, works on desktop and mobile
- No build step, no dependencies to install — just one HTML file

## Sections

- **About** — background and core skills
- **Experience** — Bharuwa Solutions internship, Google Gemini Student Ambassador role
- **Projects** — Fog-Based Flood Prediction & Early Warning System, Driver Drowsiness Detection, Online Medicine Management Web App
- **Education & Certifications** — VIT Vellore, IIT Madras, and professional certifications
- **Contact** — email, phone, LinkedIn, GitHub

## Tech

- HTML / CSS / vanilla JavaScript
- [Three.js](https://threejs.org/) (r128) via CDN for the 3D hero scene
- Fonts: Space Grotesk, Inter, JetBrains Mono (Google Fonts)

## Running locally

No build tools required. Clone the repo and open `index.html` directly in a browser:

```bash
git clone https://github.com/Aastha008/Aastha008.github.io.git
cd Aastha008.github.io
open index.html
```
